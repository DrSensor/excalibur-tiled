## New Featuers

* Parse default Tiled tmx files
  - Supports all Tiled compressions zlib, gzip, and zstd
* Parse Tile json files
* New TypeScript based object model for working with Tiled data
* Automatic Excalibur wiring for:
  * Camera
  * Colliders
  * Solid TileMap Layers

Layers:

Excalibur Layer Objects (Layer named "Excalibur")
- Type: Camera (point)
  - Property: Zoom (Default 1.0)
- Type: BoxCollider (box)
  - Property: CollisionType (Default: CollisionType.Fixed)
  - Property: ZIndex (Default: 0)
- Type: CircleCollider (ellipse)
  - Property: CollisionType (Default: CollisionType.Fixed)
  - Property: ZIndex (Default: 0)
